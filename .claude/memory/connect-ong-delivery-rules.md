Rules from the advisor site (https://willcotil.github.io/aulas2026/) for Projeto Integrador II (COTIL/UNICAMP 2026):

- **Group size:** strictly 3 or 4 members (this group has 4). No exceptions.
- **Mandatory GitHub versioning:** ALL code, database, and documentation must live on GitHub. **Evaluation considers the commit frequency of EVERY member.** Projects without version history are not accepted. ⚠️ Risk to watch: if Gabriel + Claude do all commits, the other 3 members (Luan, Abner, Arthur) need their own commit history too — flag this to the user so the team distributes/attributes work.
- **Centralized architecture + RESTful API** strongly recommended to connect all clients (already in place).
- **Three required frontends:** Web (browser, consumes API), Desktop (local Windows/Linux app, consumes API), Mobile (native or hybrid). Plus the relational DB. See [[connect-ong-architecture]] for the persona-based plan.
- **FECITEC 2026 (Aug 31 – Sep 2):** mandatory fair; project evaluated by market professionals. Deliverables: **A0 poster (PDF)** + **functional MVP**. See [[connect-ong-milestones]].

The full original project spec (Projeto Integrador.pdf) and this site were provided by the user on 2026-06-22 and should be treated as the authoritative requirements. See [[connect-ong-vision]], [[connect-ong-tech-guidelines]].
